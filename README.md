# Wave Rise LLC Website

Professional B2B eCommerce distribution website for Wave Rise LLC.

## 📁 Files

- **index.html** - Homepage with hero section and features
- **about.html** - About page with team info and mission
- **products.html** - Product catalog (36 in-stock FBA items)
- **contact.html** - Contact page with partnership CTA

## 🚀 Deployment

This site is deployed on Netlify and connected to `waverisellc.com`.

### To Update:
1. Make changes to any HTML file
2. Push to GitHub
3. Netlify auto-deploys!

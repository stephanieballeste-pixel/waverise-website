## 🚀 Deployment

This site is hosted on **Netlify** and connected to `waverisellc.com` via custom domain.

### Auto-Deploy Workflow
1. Make changes to any HTML file
2. Commit and push to the `main` branch on GitHub
3. Netlify detects the push and auto-deploys within ~60 seconds

### Manual Deploy
If needed, you can trigger a manual deploy from the Netlify dashboard under **Sites → waverise-website → Deploys → Trigger deploy**.

---

## ✏️ Making Updates

To update content on the site:

1. Open the relevant `.html` file
2. Make your edits directly in the HTML
3. Go to the GitHub repo → **Add file → Upload files**
4. Upload the updated file and click **Commit changes**
5. Netlify will automatically redeploy
